Relatório – Gincana Lógica

No dia 27/04/2026, foi realizada a aula de introdução oficial ao JavaScript com as turmas do Ensino Fundamental, avançando no cronograma do projeto LondrinenseTech. O encontro iniciou-se com a apresentação dos conceitos fundamentais da linguagem, onde o instrutor Kawan explicou a definição de variáveis e as diferentes formas de declaração, fundamentais para a construção de qualquer lógica de programação.

O foco central da atividade foi a transição do conceito para a prática. Os alunos aprenderam como cada tipo de variável funciona e a sintaxe correta para suas escritas dentro de um ambiente de desenvolvimento. Após essa etapa de codificação assistida, os estudantes realizaram uma avaliação formal para validar a compreensão dos conceitos teóricos. Ao final, para consolidar o aprendizado, foi aplicado um quiz interativo via Kahoot, permitindo que os alunos fixassem o conteúdo de forma dinâmica e competitiva.

Conclusão

A aula marcou um passo importante no projeto ao introduzir a interatividade através do JavaScript. A combinação de explicações práticas com ferramentas de gamificação mostrou-se eficiente para manter o engajamento da turma, garantindo que os alunos compreendessem a importância da estrutura de dados antes de evoluírem para scripts mais complexos.
