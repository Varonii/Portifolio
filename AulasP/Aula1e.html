Projeto Prático: Página "Sobre Mim"

Nessa semana fui chamado para ser monitor em outra turma do PensamentoComputacional. A aula começou com o instrutor Felipe apresentou os conceitos de HTML e estruturação semântica, explorando elementos básicos e técnicas de estruturação de páginas web. Com o objetivo de reforçar o aprendizado, os alunos foram desafiados a desenvolver um projeto bem prático: criar uma página no estilo "Sobre Mim", funcionando como um portfólio pessoal básico. A atividade exigia a aplicação obrigatória de tags essenciais, como <h> para o título principal, <p> para a biografia, <img> para fotos e as listas <ol> e <ul> para organizar suas habilidades.

No geral, tivemos uma aula muito boa e produtiva, já que é um tema bem prático e que eles curtem fazer. Foi bem legal acompanhar de perto eles transformando a teoria em código e compreendendo a arquitetura de um site na prática, deixando a base pronta para as próximas atividades.
