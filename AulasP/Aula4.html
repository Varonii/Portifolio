Relatório – Introdução ao HTML e CSS

Nessa aula do dia 02 de junho, os alunos começaram a entender como funciona a criação de sites com HTML, CSS e JavaScript. O instrutor Victor iniciou a explicação trazendo algumas analogias bem legais para facilitar o entendimento, mostrando que o HTML funciona como o esqueleto (estrutura), o CSS como a maquiagem (estilo) e o JavaScript como os músculos (comportamento). Logo após, ele se aprofundou nos conceitos básicos do HTML e pediu para que a turma colocasse a mão na massa criando um site próprio. Com o decorrer da atividade, o instrutor também introduziu algumas tags de CSS para que eles já fossem adicionando e estilizando suas páginas.

No geral, tivemos uma aula muito boa e produtiva, já que esse é um tema bem prático e visual. Foi bem legal acompanhar o começo deles nessa parte de desenvolvimento e ver como assimilaram rápido a diferença entre as tecnologias, deixando a base pronta para os próximos projetos.
